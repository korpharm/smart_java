- API는 회사에서 제공하는 class 파일을 모아놓은 것
- Ex01는 class 파일의 이름
- class 파일의 이름은 무조건 첫 글자는 대문자
- 무조건 파일 이름과 class 이름은 같아야 한다
  저장 이름=class 이름= 파일 이름

- 왼쪽 상단 package 창에서 All Classes를 클릭하면
  왼쪽 하단 클래스 창에서 모든 클래스가 나타남

  out => 표준출력 모니터를 뜻함

- 자바에서 ()는 무조건 메소드이다
             메소드란 기능, 동작, 작동, 실행을 의미한다
             메소드는 클래스 안에 존재한다
             () 앞에 있는게 메소드 이름
             main 메소드는 하나만 있어야 하고 꼭 있어야 함, start point.  main 메소드가 JVM을 호출
             main 메소드는 JVM이 해석허고 실행할때 제일 처음 호출해서 사용하는 메소드로 해당 프로그램에서 오직 
             하나만 존재 (시작위치)
